# AI Resume Skill Gap Analyzer

AI-powered web application that analyzes resumes, identifies skill gaps for job roles, recommends learning plans and alternate roles, tracks progress through quizzes, and updates resumes with newly acquired skills.

---

## Features

- User Registration and Login  
- Resume Upload (PDF and TXT)  
- AI Skill Extraction  
- Skill Gap Analysis  
- Match Percentage Calculation  
- Learning Plan with Deadlines  
- Quiz-Based Skill Verification  
- Alternate Role Recommendation  
- Updated Resume Download  
- Dark and Light Mode Support
